# email_trail

Automagically store an audit trail of all emails stored in your Rails app.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'email_trail'
```

```sh
$ bundle
```

## Usage

#### Installation

```sh
rails g email_trail:install
rake db:migrate
```

That's it.
